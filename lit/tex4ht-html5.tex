HTML5 requires charset declaration in form of <meta charset="charset" />.
tex4ht stores the file encoding in \verb|\a:charset| or \verb|\A:charset|
macros, in form of \texttt{charset=name}. We need to add quotes around name,
which is a job for the  \verb|\use:charset| macro.

\<use charset\><<<

\def\:charset:quotes#1=#2.{
  #1="#2"%
}
\def\use:charset{%
  \expandafter\ifx\csname a:charset\endcsname\relax%
    \expandafter\:charset:quotes\A:charset.
  \else%
    \expandafter\:charset:quotes\a:charset.
  \fi
}
>>>
